# FPS Test Micro Game

This is an FPS micro game to practice implementing a kinematic character controller for FPS games.

## Gameplay loop

- Enemy waves spawn that should be defeated to progress.
- After defeating 3 waves you win!
- If your health drops to zero you lose

## FPS character details

- Kinematic character controller
    - Sprinting
    - Crouching
    - Jumping
- Weapon system
    - Shooting
    - Aiming
    - Reloading
    - Weapon switching
